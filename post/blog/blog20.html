C++ Primer Chapter3:字符串、向量和数组3.2

string对象上的操作：string::size_type类型

  • size函数返回的是一个string::size_type类型的值。

  • string及其他大多数标准库类型都定义了几种配套的类型。这些配套类型体现了标准库类型与及其无关的特性。size_type类型就是其中一种。使用时通过作用于操作符表明名字size_type是在类string中定义的。

  • 他是一个无符号类型的值，而且能足够存放下任何string对象的大小。所有用于存放string类的size函数返回值的变量，都应该是string::size_type类型的。

  • 在c++11新标准中，允许编译器通过auto或decltype来推断变量的类型：

      auto len=line.size()//len的类型是string::size_type

  • 由于size函数返回的是一个无符号整型数，所以如果在表达式中混用了带符号数和无符号数可能引发意想不到的效果。

比较string对象

  • string类定义了集中用于比较字符串的运算符。这些比较运算符注意比较string对象中的字符，并且对大小写敏感，也就是说，比较同一个字符的大写形式和小写形式是不同的。

  • 相等性运算（==和！=）分别检验两个string对象相等或不等，string对象相等意味着他们的长度相同而且所包含的字符也全都相等，关系运算符<、<=、>、>=分别检验以恶搞string对象是否小于、小于等于、大于、大于等于另一个string对象，上述这些运算符都按照(区分大小写的)字典顺序。

              1.如果两个string对象的长度不同，而且较短string对象的每个字符都与较长string对象对应位置上的字符相同，就说较短string对象小于较长string对象。
          2.如果两个string对象在某些对应位置上不一样，则string对象的比较结果就是string对象中第一个相异字符比较的结果。
          string str="Hello";
          string phase="Hello World";
          string slang="Hiya";

  • 根据规则一可判断，对象str小于对象phase：根据规则二可判断，对象slang既大于str，也大于phase。

为string对象赋值

  • 在设计标准库类型的时候都力求在易用性上向内置类型看齐，因此大多数库类型都支持赋值操作，对于string类而言，允许把一个对象的值赋给另一个对象。

              string s1="hello,",s2="Word\n";
          string s3=s1+s2;//s3的内容是hello,word\n
          s1+=s2;//等价于s1=s1+s2;

字面值和string对象相加

  • 即使一种类型并非所需，我们也可以使用它，不过前提是该种类型可以自动转化成所需的类型。因为标准库允许把字符字面值和字符串字面值转换成string对象,所以在需要string对象的地方就可已使用这两种字面值来代替。

              string s1="Hello",s2="World";//在s1和s2中都没有标点符号
          string s3=s1+","+s2+'\n';

  • 当爸string对象和字符字面值集字符串字面值及字符串字面值混在一条语句中使用时，必须确保每个加法运算符的两侧的运算对象至少有一个时string；

              string s4=s1+",";//正确，把一个string对象和一个字面值相加
          string s5="hello"+",";//错误，两个运算对象都不是string
          //正确，每个加法运算符都有一个对象是string
          string s6=s1+","+"word";
          string s7="hello"+","+s2;//错误，不能把字面值直接相加。

  • s6中的表达式s1+","的结果是一个string对象，他同时作为第二个假发运算符的左侧运算对象，因此上述语句和下面的两个语句是等价的:

              string tmp=s1+",";//正确：加法运算符有一个运算对象是string
          s6=tmp+"word";//正确加法运算符有一个对象是string

  • 另一方面，s7的初始化是非法的

              string s7=("hello"+",")+s2;//错误，不能把字面值直接相加。

  • 因为某些历史原因，也为了与C兼容，所以c++原因呢中的字符串字面值并不是标准库类型string的对象，切记，字符串字面值与strint是不同的类型。

处理string对象中的类型

  • 我们经常需要单独处理string对象中的字符，比如检查一个string对象是否包含空白，或者把string对象中的字母改成小写，再或者查看某个特定的字符是否出现等。
  • 处理这类问题的关键是如何获取字符串本身。有时候需要处理string对象中的每一个字符，另外一些时候只需处理某个特定的字符，有时候遇到某个条件就需要停止。
  • 另一个关键问题是要知道能改变某个字符的特性，在cctype头文件中定义了一组标准库函数处理这部分工作。

处理每个字符？使用基于范围的for语句

  • 如果相对string对象中的每个字符做点儿什么操作，使用c++11新标准提供的一种语句:范围for(range for)语句。这种语句便利给定序列中的每个元素并对序列中的每个值执行某种操作，其语法形式是：

              for(declaration:expression)
          {
                  statement;
          }

  • 其中，expreession部分是一个对象，用于表示一个序列。leclaration部分负责定义一个变量，该变量将被用于访问序列中的基础元素，每次迭代，declaration部分的变量会被初始化为expression部分的下一个元素值。

  • 一个string对象表示一个字符的序列，所以string独享可以作为范围for语句的expression部分。我们可以使用范围for语句把string对象中的每个字符打印出来。

              string str("some string")
          //每行输出str中的一个字符
          for(auto c:str)//对于string中的每个字符
          {
                  cout<<c<<endl;//输出当前字符，后面紧跟换行符。
          }

  • for循环把变量c和str联系了起来，其中我们定义循环控制变量的方式与定义任意一个普通变量一样的。

  • 举个稍微复杂的例子，适用范围for语句和ispunct语句来统计string对象中标点符号的个数。

              string s("Hello World!!!");
          //punct_cnt的类型和s.size的类型一样
          for(auto c:s)//对于s中的每个字符
          {
                  if(ispunct(c))//如果该字符是标点符号
                  {
                          ++punct_cnt;//将标点符号的计数+1
                  }
          }
          cout<<punct_cnt<<"punctuation characters in"<<s<<endl;

  • 程序的输出结果是：3 punctuation characters in Hello World!!!

  • 这里我们使用decltype关键字声明计数变量punct_cnt，他的类型是s.size哈桑怒胡返回值的类型，也就是string::size_type。

使用范围for语句改变字符串中的字符

  • 如果想要改变string对象中字符的值，必须把循环变量定义成引用类型。所谓引用哥只是给定变量的一个别名，这个变量实际上被依次绑定到了序列的每个元素上。使用这个引用我们就能改变他所绑定的字符。

  • 假设我们想要将字符串改编成大小写字母的形式，可以使用标准库函数toupper，该哈桑怒胡接受一个字符，输出其对应的大写贰形式。这样，可以把整个string对象转换成大写，只要对其中的每个字符调用toupper哈桑怒胡并将结果赋给元字符就可以了：

              string s("Hello World!!!");
          //转换成大写形式
          for(auto &c:s)//对于s中的每个字符(c是引用)
          (
                  c=toupper(c);//c是一个引用，因此赋值语句见过改变s中字符的值
          )
          cout<<s<<endl;
          上述代码的输出结果将是：HELLO WORLD!!!。

只处理一部分字符？

  • 有时我们只需要访问string对象中的一个字符，或者访问多个字符但遇到某个条件就要停下来。

  • 例如，仅仅将string对象中的第一个字符或第一个单词大写化。要想访问string中的单个字符有两种方式：一种是使用下标另外一种是使用迭代器。迭代器将在后面讲到。

  • 下标运算符([])接收的参数是string::size_type类型的值，这个参数表示要访问的字符的位置。返回值是该位置闪光字符的引用。

  • string的下标从0开始，如果string对象包含至少两个字符，则s[0]是第一个字符、s[1]是第二个字符、s[s.size()-1]是最后一个字符。

  • string对象的下标必须大于等于0，而小于s.size()。使用超出此范围的下表会造成意想不到的后果。所以使用下表访问空string也会造成不可预知的效果。

  • 下标的值称为“下标”或者“索引”,任何表达式只要他的值是一个整型值，就能作为索引，如果某个索引是带符号类型的值就会自动转换成由string::size_type表达的无符号类型。

  • 使用下标运算符输出string对象的第一个字符：

              if(!s.empty())//确保有字符需要输出
          {
                  cout<<s[0]<<endl;//输出s的第一个字符
          }

  • 只要对string对象使用了下标，就要确保那个位置上确实有值。

  • 只要字符串不是常量，就能为下标运算符返回的字符赋新值。下面的程序将字符串的首字符转换成了大写形式。

              strinf s("some string");
          if(!s.empty())
          {
                  s[0]=toupper(s[0]);//为s的第一个字符赋新值。
          }
          程序的输出结果是Some string

使用下标执行迭代

  • 把s中的第一个词改成大写：

              //一次处理s中的字符直至我们处理完全部字符或者遇到一个空白符
          for(decltype(s.size()) index=0;index!=s.size()&&!isspace(s[index]);++index)
          {
                  s[index]=toupper(s[index]);//将当前字符改成大写形式
          }

  • 程序的输出结果是：SOME string

  • 逻辑运算符(&&)：c++规定，只有当左侧运算对象为真时才会检查右侧运算对象的情况。

      * 注意检查下标的合法性：使用下标时必须确保其在合理范围内.一种简便易行的方法是，总是设下标的类型为string::size_type，因为此类形式无符号数，可以确保下标不会小于0。c++标准并不要求标准库检测下标是否合法。一旦使用了一个超出范围的下标就会产生不可预知的结果。

使用下标执行随机访问

  • 其实能通过计算得到某个下标值，然后直接获取对应位置的字符，并不是每次都的从前往后一次访问。

  • 例如，想要编写一个程序把0到15之间的10进制数转换成对应的十六进制形式，秩序初始化一个字符串林奇存放16个十六进制“数字”:

              const string hexdigits="0123456789ABCDEF";//可能的十六进制数字
          cout<<"Enter a series of number between 0 and 15"<<"separated by spaces.Hit ENTER when finished:"<<endl;

          string result;//用于保存十六进制的字符串
          string::size_type n;//用于保存从输入流读取的数
          while(cin>>n)
          {
                  if(n<hexdigits.size())
                  {
                          result+=hexdigits[n];//得到对应的十六进制数字
                  }
          }
          cout<<"Your hex number is:"<<result<<endl;
          假设输入的内容为：12 0 5 15 8 15
          输出的结果为：Your hex number is C05F8F

  • 在上面的程序中下表n是string::size_type类型。也就是无符号类型。实际使用时，还要检查n是否小于hexdigits的长度。
